# drafts/

작성 중인 글이 머무는 곳입니다.

새 글은 항상 이 폴더에서 시작합니다. `templates/post-template.md`를 복사해서 만들거나, Claude Code에서 `/new-post` 명령으로 자동 생성합니다.

파일명 형식은 `YYYY-MM-DD-slug.md`이며, 여기서 `YYYY-MM-DD`는 작성 시작일(`date_created`)입니다. 발행 시점에 파일명을 발행일로 바꾸지 않아도 됩니다. frontmatter의 `date_published`가 발행일을 별도로 추적합니다.

## 초안 업데이트와 직접 작성 표시

작성 중인 `YYYY-MM-DD-slug.md` 초안은 사용자가 직접 조사한 내용을 읽고 업데이트할 수 있습니다. 사용자가 따로 정리한 자료나 메모를 전달하면 Claude Code가 그 내용을 읽고 초안에 반영합니다.

다만 사용자가 직접 작성하거나 업데이트한 항목은 `<pre></pre>` 태그로 감쌉니다. Claude Code는 `<pre></pre>` 태그 안의 내용은 확인만 하고 직접 수정하지 않습니다. 이 태그는 "이 부분은 사람이 직접 쓴 내용"이라는 표시이며, 본문 문장을 AI가 작성하지 않는다는 저장소의 핵심 원칙을 지키기 위한 장치입니다.

발행이 완료되면 이 폴더의 파일을 `posts/`로 옮깁니다. `git mv` 명령으로 옮기면 이력이 보존되어 나중에 글의 변화 과정을 추적할 수 있습니다.
